# Railway-management-system
The Railway Reservation System is implemented using Python and MySQL database management system for handling train bookings. It enables administrators to handle train information, while users can register, search, book, and cancel their tickets using this system.

## 🌟 Project Overview
Developed as a comprehensive Computer Science project, this system provides a dual-interface platform to manage the lifecycle of railway operations. It integrates a Python frontend for user interaction with a MySQL backend for high-integrity data storage.


## 🛠️ Technical Requirements

To run this system, ensure your environment meets the following hardware and software specifications:

1. Hardware: Desktop or Laptop
 
2. Software: * Python (Latest Version)
  
3. MySQL Server Python Connector Module
    
4. Tabulate library (for grid-based output)


 ## 🚀 Key Features🛡️

Administrative Empowerment (Admin Panel)

1. Dynamic Scheduling: Add new trains with detailed logistics including origin, destination, and distance.
   
2. Inventory Management: Update seat availability and adjust fares for AC, Sleeper, and General classes to meet demand.
   
3. Operational Flexibility: Instantly cancel trains in response to operational changes.
   
4. Secure Access: Protected by a dedicated admin password verification system.🙋
   
5. Customer Experience (Passenger Panel)Smart Search: Find trains instantly by entering your origin and destination.
    
6. Booking Engine: Reserve seats by selecting your preferred class and adding passenger details.
    
7. Account Safety: Secure user registration and login, including a recovery tool for forgotten User IDs via email.
    
8. Travel Flexibility: A simplified process for canceling reservations as travel plans evolve.


## 📂 Database Structure

The system relies on three core MySQL tables to maintain data integrity:

1. Train_schedule: Stores comprehensive train logs, timing, and pricing.
   
2. User: Manages passenger profiles and secure credentials.
   
3. Booked_tickets: Tracks all active reservations, PNR numbers, and ticket statuses.
